## About

Sussy Util is a TypeScript utility package that provides a set of helpful functions for performing common tasks in nodejs and web applications.

### Example usage

<CodeGroup>
```bash npm
npm i sussy-util
```

```bash yarn
yarn add sussy-util
```
</CodeGroup>

<CodeGroup>
```js JavaScript (CommonJS)
// importing everything
const sussyutil = require("sussy-util");

//importing classes/functions
const { classes } = require("sussy-util");

// importing one function/class
const { UrlUtil } = require("sussy-util");
```



```ts TypeScript / JavaScript (ES2020+)
// importing everything
import sussyutil from "sussy-util";

//importing classes/functions
import { classes } from "sussy-util";

// importing one function/class
import { UrlUtil } from "sussy-util";
```
</CodeGroup>
